### Source Citations

**What a Trading Chart Actually Shows:**
> "Most price data consists of four values per time period: Open, High, Low, Close."
[https://www.babypips.com/learn/forex/types-of-charts] [https://zerodha.com/varsity/chapter/chart-types/]

**Two Ways Charts Organize Price Data:**
> "Time-based charts create a new bar at fixed time intervals... Price-based charts create a new bar only when price moves a set distance."
[https://www.tradingsetupsreview.com/10-types-of-price-charts-for-trading/]

**Line Charts — Strengths/Limitations:**
> "Line charts excel at showing the broader trend without visual clutter."
[https://www.babypips.com/learn/forex/trading-chart-types-strengths-weaknesses]

**Bar Charts — Anatomy:**
> "The vertical line runs from the period's low to the period's high; a left tick marks the opening price; a right tick marks the closing price."
[https://zerodha.com/varsity/chapter/chart-types/] [https://www.ig.com/en/ig-academy/the-basics-of-technical-analysis/types-of-charts]

**Candlestick — Most widely used:**
> "The candlestick chart is the most widely used format in active trading."
[https://zerodha.com/varsity/chapter/chart-types/] [https://www.liberatedstocktrader.com/compare-stock-chart-types/]

**Heikin Ashi — Formula:**
> "The open of a Heikin Ashi candle is the midpoint of the previous candle's body; the close is the average of the four OHLC values for the current period."
[https://trendspider.com/learning-center/introduction-to-chart-types/]

**Renko — Definition:**
> "A new brick appears only when price moves by a set distance... if price moves less, no new brick is added."
[https://trendspider.com/learning-center/introduction-to-chart-types/] [https://www.tradingsetupsreview.com/10-types-of-price-charts-for-trading/]

**Point & Figure:**
> "One of the oldest formats in technical analysis... using X columns for rises and O columns for declines."
[https://www.tradingsetupsreview.com/10-types-of-price-charts-for-trading/]

**Range Bar:**
> "Range Bar charts create a new bar when price moves a set range, not when a time period ends."
[https://trendspider.com/learning-center/introduction-to-chart-types/] [https://www.tradingsetupsreview.com/10-types-of-price-charts-for-trading/]
